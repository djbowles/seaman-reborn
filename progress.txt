## Codebase Patterns
- Build backend is `setuptools.build_meta` (NOT `setuptools.backends._legacy`)
- pytest-asyncio uses `asyncio_mode = "auto"` in pyproject.toml
- All stub files have docstrings with "Stub - implementation in US-XXX"
- Package installed in editable mode: `pip install -e ".[dev]"`
- Quality gates: `ruff check src/ tests/` + `python -m pytest tests/ -x --tb=short` + `python -m seaman_brain --version`
- Use `datetime.UTC` not `timezone.utc` (ruff UP017 rule)
- Run ruff via `python -m ruff` (not bare `ruff`) — bare command not on PATH
- Cloud LLM providers (openai, anthropic) are in `[cloud]` optional deps: `pip install -e ".[cloud]"`
- Anthropic API separates system messages from conversation — pass via `system` kwarg, not in `messages` array
- LanceDB async: `connect_async()`, `table.search()` is a coroutine (await it), then chain `.limit(k).to_list()`
- LanceDB `list_tables()` returns a response object — access `.tables` for the list of names (replaces deprecated `table_names()`)
- LanceDB stores timestamps as ISO strings, reconstruct with `datetime.fromisoformat()`
